# Distributed System
Instant distributed system with ReactJS, Redux, NodeJS and MongoDB

# How to run
### Backend and Databse
1. Install NodeJS via [NodeJS site](https://nodejs.org/en/) . After install, you can check NodeJS on terminal by enter command `node -v` 
![NodeJS](https://i.imgur.com/3sI6I7m.png)
2. Install MongoDB on your local via [MongoDB site](https://www.mongodb.com/download-center/community) or you can do like tutorial of [MongoDB Documents](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows/)
3. Run command `mongo` on terminal ![Mongo](https://i.imgur.com/2Lhqu53.png)
4. Add `storeDB` database and `messages` collection on your mongoDB. You can use **Mongo Compass** to add more easy
5. Run `yarn install` on backend folder
6. Run `yarn start` on backend folder

### Frontend
1. Run `yarn install` on frontend folder
2. Run `yarn start` on front end folder
